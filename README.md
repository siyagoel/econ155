# PES (Payment for Ecosystem Services) Dataset Analysis

This repository contains analysis of the PES dataset with various visualizations.

## Visualizations

1. Programs by Region/State (Bar Chart) - using Name of Land Unit
2. Programs by Type of Land Unit (Bar Chart) - using Type of Land Unit
3. Programs by Themes Analyzed (Bar Chart) - using Themes Analyzed
4. Programs Over Time (Line Chart)
5. Ecological Effectiveness Assessment
6. Ecological Effect Direction
7. Social Effectiveness Assessment
8. Social Effect Direction

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Open the Jupyter notebook:
```bash
jupyter notebook PES_analysis.ipynb
```

## Data

The analysis uses the `PESMEX20_Database-and-Definitions_06-24 (2).xlsx` file.

## Requirements

- pandas
- numpy
- matplotlib
- seaborn
- openpyxl
